# Versioning

All official projects in this repository start at version `1.0`.

From this point:

- A smaller compile/release increments the app version by `0.1`.
- A larger release requires asking whether it should be treated as final.
- If a larger release is final, increment the major version by `1.0`.

Local working copies are kept in separate version folders under
`Desktop/github/Esp32-WiFi-Radio/`, for example `V1.0`, `V1.1`, and `V2.0`.

Keep the visible version strings in firmware, README files, changelog, and tags
aligned before publishing.

Every published version should have a matching GitHub release.
